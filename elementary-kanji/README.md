# clasp ts

GAS の CLI ツール「clasp」で TypeScript を扱うための雛形

## Prerequred

- @google/clasp

## Development

1. module install
    ```bash
    npm install
    ```
2. clasp create
    ```bash
    $ clasp create --title ${TITLE} --rootDir src
    ```

## Memo

- GAS から clone
    ```bash
    $ clasp clone ${scriptId}
    ```
- GAS へ push
    ```bash
    $ clasp push
    ```
